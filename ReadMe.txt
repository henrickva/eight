Descrição

O projeto "Eigth's" é uma aplicação é uma rede social que permite a escrita, 
edição e compartilhamento de postagens online. Foi criado com Next.js, uma estrutura de 
desenvolvimento web baseada em React, para oferecer uma experiência 
de usuário rápida e eficiente.

Principais Características

    Gerenciamento de Postagens: Os usuários podem criar, editar e excluir postagens 
diretamente no painel de controle.

    Conteúdo Dinâmico: As postagens são renderizadas de forma dinâmica, permitindo 
que os visitantes do blog leiam as postagens mais recentes sem a necessidade de recarregar a página.

    Comentários e Feedback: Os leitores podem deixar comentários nas postagens para
iniciar discussões e interagir com o autor.

    Autenticação de Usuário: Um sistema de autenticação permite que os autores façam
login e gerenciem suas postagens de forma segura.

    Design Responsivo: O projeto é totalmente responsivo, adaptando-se a dispositivos 
móveis, tablets e desktops para proporcionar uma experiência de usuário consistente.

Tecnologias Utilizadas

O projeto faz uso das seguintes tecnologias:

    Next.js: Como estrutura principal para o desenvolvimento front-end.

    Material UI: Para montar o design da página.

    NextAtuh: Para autenticar usuários e fazer o login social.
